# EventWeb
練習寫網頁，有緣再改  
用到Vue+Express，參考https://blog.csdn.net/u012414590/article/details/79043757  
node_modules資料夾很大所以就不上傳了 


# 基本環境
npm install -g vue-cli  
vue init webpack EventWeb  
cd EventWeb  
npm install  
npm install express --save  

# 其他
DataPicker拿vue2-datepicker修改  

# 打包
npm run build  
node server/app.js
